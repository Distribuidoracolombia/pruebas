Descripción del Proyecto

Este proyecto integrador evaluará sus habilidades combinadas en Excel. Deberá crear un dashboard interactivo que analice datos de ventas y presente información relevante para la toma de decisiones.

Instrucciones

  1. Descargue el archivo de datos proporcionado.
  2. Cree un dashboard que incluya al menos 3 gráficos diferentes.
  3. Implemente filtros interactivos usando segmentación de datos.
  4. Cree al menos 2 tablas dinámicas para analizar los datos.
  5. Utilice fórmulas avanzadas para calcular KPIs relevantes.
  6. Asegúrese de que el dashboard tenga un diseño profesional y sea fácil de usar.

Criterios de Evaluación

  • Funcionalidad (40%): Correcto funcionamiento de todas las características solicitadas.
  • Análisis de datos (30%): Calidad y relevancia de los análisis presentados.
  • Diseño y usabilidad (20%): Apariencia profesional y facilidad de uso.
  • Creatividad (10%): Elementos adicionales que mejoren el dashboard.
